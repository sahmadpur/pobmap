## Transport Vehicle Assets

The moving transport markers on the public map use preview renders derived from public-domain Poly Pizza models.

- `public/transport-assets/truck.png`
  Source: [Truck - Free 3D Model By Quaternius](https://poly.pizza/m/cXw6oiFtZ8)
  License: Public Domain (CC0)
- `public/transport-assets/train.png`
  Source: [Locomotive Front - Free 3D Model By Quaternius](https://poly.pizza/m/WY84FHug9s)
  License: Public Domain (CC0)
- `public/transport-assets/ship.png`
  Source: [Ship - Free 3D Model By Kenney](https://poly.pizza/m/SPxFN3Oazd)
  License: Public Domain (CC0)

The `*-source.jpg` files in the same directory are downloaded source preview renders from Poly Pizza. The `*.png` files are keyed and cropped local derivatives prepared for use as map animation sprites.
